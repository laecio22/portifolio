## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/laecio22/portifolio.git
```

Entre no diretório do projeto

```bash
  cd portifolio
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
